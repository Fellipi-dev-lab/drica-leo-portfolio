# 🏠 Drica & Léo - Personal Organizer


Site profissional desenvolvido para **Drica & Léo**, serviço de personal organizer em Recife-PE.

🔗 **Link do site:** [https://site-drica-leo-eight.vercel.app](https://site-drica-leo-eight.vercel.app)

---

## 📋 **Sobre o Projeto**

Site institucional completo com 5 páginas, desenvolvido para apresentar os serviços de organização profissional, com foco em experiência do usuário e design atraente.

### 🎯 **Objetivo**
Criar uma presença online profissional que transmita confiança e mostre os resultados do trabalho de organização, facilitando o contato de potenciais clientes.

---

## 🚀 **Tecnologias Utilizadas**

| Tecnologia | Para quê |
|------------|----------|
| **React.js** | Biblioteca principal para construção da interface |
| **Vite** | Build tool para desenvolvimento rápido |
| **React Router DOM** | Gerenciamento de rotas e navegação |
| **CSS3** | Estilização e design responsivo |
| **React Player** | Player de vídeo para galeria multimídia |
| **Git/GitHub** | Versionamento e hospedagem do código |
| **Vercel** | Deploy e hospedagem do site |

---

## ✨ **Funcionalidades Implementadas**

✅ **5 páginas completas:** Home, Serviços, Sobre, Galeria e Contato  
✅ **Galeria interativa** com vídeos e imagens organizadas por categoria  
✅ **Filtros dinâmicos** para facilitar a navegação na galeria  
✅ **Formulário de contato funcional** integrado com FormSubmit  
✅ **Links diretos** para WhatsApp e Instagram  
✅ **Botão "Voltar ao topo"** em todas as páginas  
✅ **Selo "Profissional Certificado"** flutuante  
✅ **Design 100% responsivo** para celular, tablet e desktop  
✅ **Otimização de imagens** para carregamento rápido
